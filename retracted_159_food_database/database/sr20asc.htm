SR20 Data Files - ASCII

This directory contains the full dataset. The data files are ASCII delimited and uncompressed. Fields are separated by the caret (^) symbol, and character fields are enclosed by tilde (~) marks. The files are:

  • Food Description (File Name = FOOD_DES.txt)
  • Food Groups (File Name = FD_GROUP.txt)
  • Nutrient Data (File Name = NUT_DATA.txt)
  • Nutrient Definitions (File Name = NUTR_DEF.txt)
  • Weight (File Name = WEIGHT.txt)
  • Derivation Code (File Name - DERIV_CD.txt)
  • Source Code (File Name = SRC_CD.txt)
  • Footnotes (File Name = FOOTNOTE.txt)
  • Sources of Data (File Name = DATA_SRC.txt) - This file is also available as a PDF file.
  • Data Source Link (File Name = DATSRCLN.txt)

Documentation is available as a PDF (SR20_doc.pdf) file.  It explains the relationships among the data files and provides the format for each.

These files are in the directory [CD-ROM Drive]:\SR20DATA\ASCII
